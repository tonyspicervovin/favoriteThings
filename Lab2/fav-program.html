Media Manager

My favorite program to date

What is it?

This is a program that I created for media collectors for organizing and cataloging their media

  • It allows you to add media with characteristics description, price paid, media type and condition
  • It stores your media in a database and uses that database to display a table
  • It allows you to edit within the table as well as delete the selected media
  • has query options such as a search function and show all of a certain type
